# Work From Home Burnout Predictor 

This project uses Machine Learning to predict burnout levels for remote workers based on work habits.

## Features
✔ Predict burnout score  
✔ Predict burnout risk (Low / Medium / High)  
✔ Interactive Streamlit interface  
✔ Multiple ML models used  

## Technologies
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Run the Project

pip install -r requirements.txt

streamlit run app.py
